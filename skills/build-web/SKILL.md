# Build Web

Use `explore-ecosystems` for material dependencies. Classify the surface as
content-first, application-first, or hybrid.

1. Map routes, layouts, islands, server functions, queries, URL state, forms,
   and design-system ownership.
2. Use renderer-native semantics; never mechanically translate React lifecycle
   or state patterns into Solid.
3. Compose existing primitives before modifying design-system internals.
4. Distinguish router search state from server-cache state.
5. Verify keyboard/focus behavior, accessible names, all data states, responsive
   layout, and performance in a browser.
